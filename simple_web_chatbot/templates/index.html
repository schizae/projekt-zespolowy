O Projekcie

W wersji pierwszej był użyty lokalny model LLM Neo Gpt 1,7T, ale był zbyt głupi.

Kolejna wersja korzystała z mocniejszego ale dalej starego modelu lokalnego co powodowało przeciązenia laptopa.

Wiec postanowilismy na webowa wersje API od Open AI, nastepnie zostało przygotowane UI oraz strony poboczne i inne funkcjonalności.


Ta wersja korzysta, z API GPT 4 oraz ich chmury obliczeniowej.
